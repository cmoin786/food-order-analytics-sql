# Food Delivery SQL Analytics Project

## Project Overview

This project analyzes food delivery operations using SQL. The dataset contains information about customers, restaurants, delivery agents, orders, and order items.

The project demonstrates database design, data loading, SQL querying, business analysis, and advanced SQL concepts such as CTEs and window functions.

## Database Schema

Tables:

* Customers
* Restaurants
* Delivery Agents
* Orders
* Order Items

## Skills Used

* PostgreSQL
* Database Design
* Data Modeling
* Joins
* Aggregations
* CTEs
* Window Functions
* Business Analytics

## Business Questions Solved

### Customer Analysis

* Total customers
* Repeat customers
* Highest spending customers
* Customer lifetime value

### Revenue Analysis

* Total revenue
* City-wise revenue
* Restaurant-wise sales
* Revenue percentage contribution

### Restaurant Analysis

* Top restaurants by revenue
* Restaurant ranking
* Average order value
* Top-selling food categories

### Delivery Analysis

* Fastest delivery agent
* Average delivery time
* Late deliveries
* Delivered vs Cancelled orders

### Advanced SQL

* Running revenue totals
* Revenue growth analysis
* Ranking functions
* Lag functions
* Common Table Expressions (CTEs)

## Key Insights

* Identified top-performing restaurants.
* Measured customer lifetime value.
* Evaluated delivery performance.
* Analyzed payment preferences.
* Generated city-wise and restaurant-wise revenue insights.
